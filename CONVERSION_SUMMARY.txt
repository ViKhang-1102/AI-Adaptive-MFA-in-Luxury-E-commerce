================================================================================
                          IMPLEMENTATION NOTES
================================================================================

NUMBER FORMAT RATIONALE:
Vietnamese Dong (VND) is an integer-only currency. Unlike USD with cents,
VND doesn't use fractional units, so displaying prices without decimal 
places is the correct approach.

Examples:
- 1,500 VND (not 1,500.00 or 15.00)
- 29,990,000 VND (not 29,990,000.00)
- 199,000 VND (not 199,000.00)

SYMBOL PLACEMENT:
The ₫ symbol is placed before the amount, which is standard for Vietnamese
currency display:
- ₫29,990 (correct)
- Not: 29,990 ₫

DATABASE PRESERVATION:
All existing database records remain unchanged. The conversion only affects
how prices are DISPLAYED to users, not how they're stored. A product priced
at 29990000 in the database displays as "₫29,990,000" on the frontend.
